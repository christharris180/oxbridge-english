# QUICK START - Deploy in 5 Minutes!

## 🚀 Super Fast Deployment

### Step 1: GitHub (2 minutes)
1. Go to https://github.com → Sign up/Login
2. Click **"+"** → **"New repository"**
3. Name it: `oxbridge-english`
4. Make it **Public**
5. Click **"Create repository"**
6. Click **"uploading an existing file"**
7. **Drag ALL your website files** into the browser
8. Click **"Commit changes"**

### Step 2: Cloudflare Pages (3 minutes)
1. Go to https://pages.cloudflare.com → Sign up/Login
2. Click **"Create a project"**
3. Click **"Connect to Git"** → **"Connect GitHub"**
4. Select your **oxbridge-english** repository
5. Click **"Begin setup"**
6. Leave all settings as default
7. Click **"Save and Deploy"**
8. **Wait 2 minutes** ⏱️
9. **DONE!** 🎉

### Your Live Site:
```
https://oxbridge-english.pages.dev
```

## ⚠️ Important Notes

**Video File Sizes:**
- GitHub has 100 MB limit per file
- If your videos are larger, you have 2 options:
  1. Compress them (use Handbrake, free)
  2. Use Git LFS (see full guide)

**After Deployment:**
- Site updates automatically when you push to GitHub
- Changes appear in 1-2 minutes
- Free forever, unlimited bandwidth!

## 🆘 Need Help?

**Problem:** Video files too large for GitHub
**Solution:** See DEPLOYMENT-GUIDE.md section "Troubleshooting"

**Problem:** Site not updating
**Solution:** Wait 2 minutes, clear browser cache (Ctrl+Shift+R)

**Problem:** Email obfuscated
**Solution:** Cloudflare Dashboard → Scrape Shield → Email Obfuscation → OFF
