Course 1

  • Form correct sentences in English from the very first lesson.
  • Understand key A1-level grammar: present simple, articles, pronouns, and more.
  • Gain confidence speaking and writing for travel, work, or study.
  • Practice with interactive exercises designed específicamente para hispanohablantes nativos.

Course 2

  • Reinforce vocabulary and structures from the previous course.
  • Use have/has got para expresar posesión.
  • Use object pronouns correctamente.
  • Introduce and practice adverbs of frequency.
  • Understand and use prepositions of place and time.
  • Describe environments using there is / there are.

Course 3

  • Use the present continuous correctly to describe actions happening now.
  • Express possibility or permission using can.
  • Talk about rules and responsibilities using have to.
  • Avoid repetition using one and ones.
  • Introduce the past simple to describe past events or experiences.

Course 4

  • Understand and use future verb tenses correctly.
  • Identify and apply the different uses of like in context.
  • Expand vocabulary to talk about plans, goals, and opinions.
  • Improve fluency in conversations about the present, past, and future.
  • Prepare to successfully progress to the intermediate (B1) level.
